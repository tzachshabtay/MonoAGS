    Class ButtonAnimation

    Allows changing properties of the button to "animate" it as it moves between states (idle, hover, pushed). Each ButtonAnimation represents one of this states, and can have a different animation, border, background color (tint) or text rendering configuration (font, shadows, etc- ITextConfig).
